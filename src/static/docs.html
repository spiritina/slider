Parametrs

Autoplay

Parametr at settings: autoplay: boolean,

Autosetting: true,

Description: if true, autostarts slides changing in foreward and then in backward direction.

Arrows

Parametr at settings: arrows: boolean,

Autosetting: true,

Description: if true, unabled the left and right arrows.

Dots

Parametr at settings: controls: boolean,

Autosetting: true,

Description: if true, unabled the dots in the bottom of the slider.

Draggable

Parametr at settings: draggable: boolean,

Autosetting: false,

Description: if true, unabled changing the slides by dragging them by mouse.

Key Control

Parametr at settings: keyControls: boolean,

Autosetting: false,

Description: if true, unabled changing the slides using arrows keys on the keyboard.

Interval

Parametr at settings: interval: integer,

Autosetting: 2000,

Description: time interval in miliseconds between slide changing during autoplaying.

Methods

Autoplay Start

Function: yourGalleryObject.autoplayStart()

To start slides autoplaying.

Autoplay Stop

Function: yourGalleryObject.autoplayStop()

Description:To stop slides autoplaying.

Go to chosen slide

Function: yourGalleryObject.goToSlide(int)

Description:To change the slide to chosen, numeration starts from 0.

Initialize

yourGallery = new Gallery(settings)

Function: yourGalleryObject.goToSlide(int)

Description:To change the slide to chosen, numeration starts from 0. Settings are optional.

To the next slide

Function: yourGalleryObject.slideNext()

Description:To change to the next slide, if it's posible. No loop changing.

To the previous slide

Function: yourGalleryObject.slidePrev()

Description:To change to the previous slide, if it's posible. No loop changing.
